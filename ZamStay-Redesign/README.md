# ZamStay Property Listing Redesign

## Project Structure
- `index.html` - Main HTML file
- `styles.css` - CSS styles
- `script.js` - JavaScript functionality
- `Launch-ZamStay.ps1` - PowerShell launcher script

## Features Implemented
1. **Vertical Stack Layout** - Full-width property cards stacked vertically
2. **Category Filter Navigation** - Apartments, Hotels, Lodges filtering
3. **Property Card Order**:
   - Image with price overlay
   - Dynamic amenities based on category
   - Integrated mini-map per property
   - Local attractions list
   - Book Now button at bottom
4. **Clean Modern Design** - Plenty of white space, modern UI

## How to Run
1. Run `Launch-ZamStay.ps1` in PowerShell
2. Or open `index.html` directly in a browser

## Integration Notes
Replace sample property data in `script.js` with your actual API calls.
